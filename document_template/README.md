# Templates Oficiais FLG

> Estes são os templates oficiais da Founders Led Growth. Não edite diretamente — use-os como base copiando para a pasta do cliente.

---

## Templates Disponíveis

| Template | Caminho | Descrição |
|----------|---------|-----------|
| Slides | `slides/template.html` | Apresentação interativa com navegação por teclado |
| Documento | `documento/template.html` | Documento estratégico completo para leitura |

## Assets Compartilhados

| Arquivo | Descrição |
|---------|-----------|
| `assets/logo-flg.svg` | Logo FLG vetorial (branco, para fundo escuro) |
| `assets/logo-flg.png` | Logo FLG rasterizado |

## Guias

| Guia | Descrição |

## Regra de Ouro

**Nunca edite os templates diretamente.**
Use sempre: `./scripts/new-client.sh` para criar uma cópia para o cliente.
