## Hash tracking

Quando este arquivo for completado, o hash SHA-256 dos primeiros 1000 chars
será embutido em todas as chamadas Claude que usarem este prompt
(via Anthropic API metadata). Isso permite rastrear qual versão do prompt
gerou cada output.
